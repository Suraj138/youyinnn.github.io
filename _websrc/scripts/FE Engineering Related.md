### Prettier

#### Setting up on vscode

1. vscode settings:

   ``` javascript
   {
       "editor.formatOnSave": true,
       "editor.defaultFormatter": "esbenp.prettier-vscode",
   }
   ```

2. Install extension

3. Configurations

   https://prettier.io/docs/en/options.html

   e.g

   ``` javascript
   {
     "singleQuote": true,
     "arrowParens": "avoid"
   }
   ```
