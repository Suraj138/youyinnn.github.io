一顿操作 node.js安装

🔹安装nvm

  1. 首先去 https://github.com/coreybutler/nvm-windows 下载最新的release版本 然后安装即可

🔹使用mvn安装nodejs的多版本

$ nvm install 0.10
$ nvm install 4
$ nvm install 6
$ nvm install 8

🔹安装全局npm

可以看到这里安装了4种版本的nodejs, 每个版本都带有自己的npm, 如果要特定使用npm, 则需要先输入

mvn use 8

比如上面先use了8版本的node

然后再安装全局npm, 不然你找不到npm的环境
